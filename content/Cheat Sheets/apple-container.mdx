## Installation & Setup

### Requirements

```bash
# Apple silicon is required
uname -m                          # arm64

# Apple's project currently supports macOS 26+
sw_vers
```

### Install Apple Container

```bash
# Open the official releases page and download the signed .pkg installer
open https://github.com/apple/container/releases

# After installation, verify the CLI is available
container system version
```

### Start the Service

```bash
container system start
container system status
container system version
```

### Upgrade, Downgrade, Uninstall

```bash
# Stop services before changing versions
container system stop

# Upgrade to the latest release
/usr/local/bin/update-container.sh

# Install a specific release
/usr/local/bin/update-container.sh -v <version>

# Uninstall but keep user data
/usr/local/bin/uninstall-container.sh -k

# Uninstall and remove user data
/usr/local/bin/uninstall-container.sh -d
```

### Useful Aliases

```bash
alias ctr='container'
alias ctrls='container ls'
alias ctri='container image'
alias ctrsys='container system'
```

## System & Service Commands

### Health, Version, Logs

```bash
container system start
container system stop
container system status
container system status --format json
container system version
container system logs
container system logs --last 1h
container system logs -f
container system df
container system property list
container system property list --format json
```

### DNS & Kernel Management

```bash
# Create a local DNS domain for containers
sudo container system dns create dev.local
container system dns list
sudo container system dns delete dev.local

# Install or refresh the recommended kernel
container system kernel set --recommended
```

## Container Lifecycle

### Run Containers

```bash
container run -it ubuntu:latest /bin/bash
container run --rm alpine:latest echo "hello"
container run -d --name web -p 8080:80 nginx:latest
container run -e NODE_ENV=production --cpus 2 --memory 1G node:20
container run -v "$PWD":/workspace -w /workspace ubuntu:latest bash
container run --ssh -it ubuntu:latest bash
container run --rosetta --platform linux/amd64 ubuntu:latest bash
```

### Create, Start, Stop, Remove

```bash
container create --name job ubuntu:latest sleep 300
container start -a job
container stop job
container stop --all
container kill job
container rm job
container rm --all
container prune
```

### List, Inspect, Logs, Exec

```bash
container ls
container ls -a
container ls --format json
container inspect web
container logs web
container logs -f -n 100 web
container exec -it web /bin/sh
container exec -e DEBUG=1 -w /app web ./bin/start
container stats
container stats --no-stream web
container stats --format json --no-stream web
```

### Copy Files & Export Filesystems

```bash
# Copy host -> container
container cp ./config.yaml web:/etc/app/

# Copy container -> host
container cp web:/var/log/nginx/access.log ./logs/

# Export a stopped container filesystem
container stop web
container export -o web.tar web
```

## Image Build & Management

### Build Images

```bash
container build -t my-app:latest .
container build -f Dockerfile.prod -t my-app:prod .
container build --target production --no-cache -t my-app:prod .
container build --build-arg NODE_ENV=production -t my-app .
```

### Image Commands

```bash
container image ls
container image pull alpine:latest
container image inspect alpine:latest
container image tag my-app:latest registry.example.com/team/my-app:latest
container image push registry.example.com/team/my-app:latest
container image save -o my-app.tar my-app:latest
container image load -i my-app.tar
container image rm my-app:latest
container image prune
```

### Registry Authentication

```bash
container registry login registry.example.com
container registry list
container registry logout registry.example.com
```

## Networks, Volumes & Builder

### Networks

```bash
container network create app-net
container network create --subnet 192.168.100.0/24 app-net
container network ls
container network inspect app-net
container run -d --name api --network app-net nginx:latest
container network rm app-net
container network prune
```

### Volumes

```bash
container volume create mydata
container volume create -s 10G build-cache
container volume ls
container volume inspect mydata
container run -v mydata:/data alpine:latest
container volume rm mydata
container volume prune
```

### Builder

```bash
container builder start
container builder status
container builder stop
container builder rm
```

## Common Workflows

### Quick Web Server

```bash
container run -d --name web -p 8080:80 nginx:latest
container logs -f web
container exec -it web /bin/sh
container stop web
container rm web
```

### Local Development Mount

```bash
container run --rm -it \
  -v "$PWD":/workspace \
  -w /workspace \
  ubuntu:latest \
  bash
```

### Build and Push

```bash
container build -t registry.example.com/team/my-app:latest .
container registry login registry.example.com
container image push registry.example.com/team/my-app:latest
```
